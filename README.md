## Inventory program

1) This program is used to add products to an inventory, also to show the content of the inventory and the statistics of it.
2) The program uses validations to help the program don't get crashed
3) The program uses a menu to access to each option

The functions and processes executed in this programs helped to put into practice the learning objective of the week #2 that was to use conditionals, cicles and functions with an interactive menu organizing data in an inventory.
